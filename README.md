# CATCHING - THE SOCIAL APP
👥 catching is a real time social application where users can connect with each other based on their live location rendered on the map under the radius of 3 Km of their exact location

## WHAT IS THE NEED OF THIS APPLICATION 🤔❓
  🙋 imagine that you are sitting alone in a place where you can't call on your close ones to give you company. YOU END UP BEING BORED AND SOPHISTICATED.
      Catching resolves this boredom... You can hop on to this application, make your profile and find people near you who are using this in real time and send a         request to them and invite them to join you. 

### KEY FEATURES 📋
• SignUp/ LogIn
• Profile Picture 
• Personal details - name, age, gender, profession 
• Current status (0-100 letters)
• Post 
• Real time Map rendering
• Connect to people who are online under 3km radius
• Live chat window

### SECURITY FEATURES 🛡️
• JWT authentication + sessions
• Chat deletes when users go offline 
• Permission for turning on location 
• Secured data pipelines 
• End-to-End encrypted chatting
